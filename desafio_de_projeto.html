HTML, que significa 'Hiper Text Markup Language'é a linguagem padrão utilizada para a criação de páginas wed. é ulilizada para estrutura de conteúdo e apresentação visual de uma página da web.
HTML consiste em uma séries de elementos ou 'tags'que são utilizadas para parcar diferentes partes do conteúdo como textos, imagens, links e muito mais.


Algumas tags HTML

A linguagem de marcação HTML apresenta dezenas de tags que dão uma melhor funcionalidade e visibilidade as páginas web, mas neste arquivo, irei apresentar apenas algumas tags mais usadas em HTML:

Tag "font":

A tag "font" é uma tag HTML que contumava ser usada para definir a fonte, a cor e o tamanho de texto de uma página web. No entanto, o uso da tag font está desencorajado em favor do uso de CSS (Cascading style Sheets) para estilização. A principal razão para isso é que o CSS permite separar o connteúdo do estilo, tornando o código mais organizado, mais flexivel para as mudanças de design.


Tag "del":

A tag "del"é uma tah HTML utilizada para indicar que um trecho de texto demarcado foi removido, riscado ou está obsoleto. Ela é frequentemente utilizada para mostrar alterações ou revisões, ou para destacar que determinado conteúo não está mais em uso.


Tag "p":

A tag p é umas das tags HTML mais usadas na criação de sites. ela é usada para definir um parágrafos de texto.
É um das tags mais básicas e amplamente utilizada em páginas web para estruturr o coteúdo em blocos de textos separdos.


Tag "abbr":

A tag "abbr" é uma tag HTML utilizada para para definir uma abreviaçção ou sigla. Ela é geralmente usada para fornecer uma informação mais detalhada quando o usuário pssa o mouse sobre a abreviasão ou sigla.
Isso ajuda a melhorar a acessibilidade e a compreensão do contúdo, especialmente quando termos ou siglas não são conhecidos.


Exixtem outras tags HTML muito usadas mas por enquanto é tudo. Numa outra natéria traremos mais conteúdo acerca das tags HTML.
